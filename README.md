# cli

LeetCode CLI

## List of Commands

###login <username>
    ask for password

### list <tags>

### goto <tag>

### list <problems>
    --solved | --open
    --difficulty
    --frequency
    --company

### open <problem #>
    get description
    get boilerplate
    save temp file to local dir

### check <problem #>
    check syntax
    run sanity check

### submit <problem #>
    test solution
    display errors

### help <problem #>
    display solution from local file
    display best solutions from Web

### logout
