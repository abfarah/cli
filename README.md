# CLI

Start screen, vim and cli.py. Practice as many questions as you can:-)

Note: this tool is not affiliated with leetcode.com.

## Requirements

```sudo pip install requests lxml bs4 PyExecJS ascii_graph```

## Commands

```
login

chmod < language >

ls

/ < keyword >

cd < tag | problem | .. >

cat

pull

check

push

cheat < number of examples >

clear

top
```
Control+D to exit
